# message-crud
Simple CRUD application which allows to manage messages.

---

## End-points
- [POST] localhost:8080/messages/ - create new message with autogenerated id and content 'Wartosc'
- [GET] localhost:8080/messages/ - show all messages stored in database
- [PUT] localhost:8080/messages/{id} - update message with given id
- [GET] localhost:8080/messages/{id} - show message with given id
- [GET] localhost:8080/messages/random/{count} - show _count_ random messages from database

---
## Technologies:
- Spring
- JUnit
- SQLite
